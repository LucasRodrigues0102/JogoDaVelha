# Jogo da Velha

Este repositório contém um projeto de Jogo da Velha desenvolvido por mim em C#, utilizando Windows Forms, durante o segundo semestre de 2024. O projeto faz parte da minha formação acadêmica e está sendo disponibilizado neste repositório como parte do meu portfólio para busca de estágio na área de desenvolvimento de software.

O principal objetivo do projeto foi aplicar, de forma prática, os conceitos fundamentais de lógica de programação aliados ao desenvolvimento de uma interface gráfica funcional e interativa. A proposta envolveu não apenas criar o jogo visualmente, mas também implementar corretamente suas regras, garantindo uma experiência coerente para o usuário.

A aplicação permite a interação de dois jogadores, realiza a alternância correta de turnos, valida jogadas, identifica automaticamente condições de vitória ou empate e possibilita o reinício da partida ao final do jogo. Toda a lógica foi integrada à interface gráfica por meio de programação orientada a eventos, reforçando a relação entre ações do usuário e comportamento do sistema.

Embora seja um projeto de escopo simples, ele representa um passo importante no desenvolvimento das minhas habilidades como estudante de tecnologia, demonstrando a capacidade de transformar requisitos em código funcional, estruturar uma aplicação desktop em C# e organizar a lógica de forma compreensível e reutilizável.

O projeto foi desenvolvido no Visual Studio, utilizando C# e Windows Forms, e pode ser executado localmente abrindo a solução e compilando a aplicação. Trata-se de um projeto acadêmico com fins educacionais, mantido neste repositório para fins de estudo e avaliação de competências técnicas iniciais.

### Experiências adquiridas

Durante o desenvolvimento deste projeto, foi possível consolidar conhecimentos importantes para a formação como desenvolvedor, especialmente no que se refere à lógica de programação, uso de estruturas condicionais, controle de fluxo e manipulação de eventos em aplicações gráficas. Também contribuiu para o entendimento de como estruturar um projeto em C#, organizar código, lidar com estados da aplicação e pensar na experiência do usuário, mesmo em aplicações simples.

Além do aspecto técnico, o projeto reforçou habilidades como atenção a detalhes, resolução de problemas, raciocínio lógico e organização.
